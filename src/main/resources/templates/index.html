    Pour ajouter des Fichiers sur le cluster:


    Copier le fichier dans le pod:
    >> kubectl cp fichier/people.csv hadoop-hadoop-hdfs-nn-0:/ -n data
    Monter le fichier dans le cluster:
    >> hdfs dfs -put people.csv /user/
    Vérifier la présence du fichier:
    >> hdfs dfs -ls /data/
